แนวคิดที่นำมาประยุกต์ใช้

1. OOP (Object-Oriented)

การเขียนโปรแกรมเชิงวัตถุ เช่น Inheritance (สืบทอด Activity) และ Encapsulation (ซ่อนข้อมูลภายในคลาส)

2. Multitasking

ใช้ Background Thread ในการอ่านไฟล์เอกสารและเรียก API เพื่อให้ UI ของแอปทำงานได้ลื่นไหล ไม่ค้าง

3. MVC Architecture

การแยกส่วนการทำงานเป็น Model (ข้อมูล), View (หน้าจอ XML) และ Controller (คลาส Activity)

4. Data Persistence

การใช้ SQLite Database เก็บประวัติถาวร และ JSON ในการเก็บโครงสร้างข้อมูลเฉลยที่ซับซ้อน

5. API & AI Integration

การเชื่อมต่อ Cloud AI ผ่าน REST API (Retrofit) เพื่อประมวลผลข้อมูลขนาดใหญ่แบบอัจฉริยะ

Project Architecture (13 Classes with Encapsulation)
